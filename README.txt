Как использовать?
1) В верхнем окне пишем сообщение с цветной разметкой (&0 - &9 и &a - &f);
2) Нажимаем "Просмотр";
3) Если нравится появившийся вариант в нижнем окне, можно "Скопировать";
4) Если хотите очистить ОБА окна, нажмите "Очистить";

Для чего нужна?
Для предпросмотра сообщений, использующих цветную разметку. Идеально подходит для создания MOTD гильдий.

Если программа работает некорректно, попробуйте запустить от 
имени администратора.
Программа работает ТОЛЬКО на 64 битной системе.


Если вас что-то не устраивает, то вы всегда можете скачать исходники 
и похимичить с ними.
